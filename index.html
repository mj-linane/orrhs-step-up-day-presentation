AP Computer Science A

Learn how to build finely crafted apps using the Java programming language. Focus of this class is building programs using an object-oriented programming pattern. While you do not need previous experience with programming, this can be a very challenging course for beginners. In this course you will learn about computing but through the lens of designing programs from scratch in the most time and memory efficient way.

  • Photo Editing App

  • Pokemon Game

  • ChatBot

AP Computer Science P

Learn the fundamentals of computer science and build build finely crafted web apps using the JavaScript programming language. The focus of this course is to give you a solid foundation in the science and mechanics of computing. This course covers a wide range of topics that include app design, binary, the internet, algorithm design, computer memory and storage, cyber security, and machine learning.

  • Forecaster App

Web Design and Development I

In this class, you learn how to design and build beautiful websites. You will start by drafting out a vision for the website, called wireframing. Then you will learn the key programming languages of the internet to build out the site and add its functionality.

  • Tribute Page

Multimedia Design

In this course you will learn how to make your digital designs aesthetically pleasing. The course will focus on establishing strong fundamentals in the basics of graphic design. This includes the topics: colors, fonts, spacing, and item heirarchies. Then you will move on to projects like designing for social media and logos. You will then learn how to professional edit photos. Lastly, you will learn how to build scalable images and graphics. You will then learn how to do basic animations with those drawings.

  • Graphic Design / Logo Design

  • Photoediting

  • Digital Drawings
